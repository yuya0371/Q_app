# Q. プライバシーポリシー

最終更新日: 2026年2月9日

---

## 1. はじめに

Q.運営（以下「当社」といいます）は、モバイルアプリケーション「Q.」（以下「本アプリ」といいます）をご利用いただくお客様（以下「ユーザー」といいます）の個人情報の保護に努めています。

本プライバシーポリシー（以下「本ポリシー」といいます）は、当社がどのような情報を収集し、どのように利用・保護するかについて説明するものです。本アプリをご利用いただく前に、本ポリシーをよくお読みください。

## 2. 収集する情報

### 2.1 お客様が直接提供する情報

| 情報の種類 | 詳細 | 必須/任意 |
|---|---|---|
| メールアドレス | アカウント登録・認証に使用。小文字に正規化して保存 | 必須 |

| プロフィール画像 | 他のユーザーに表示されるアイコン画像。JPEG/PNGで受付、512x512にリサイズして保存 | 任意 |

| 通報内容 | 通報時の自由記述テキスト | 任意 |

## 3. 情報の利用目的

当社は、収集した情報を以下の目的で利用します。

1. **サービスの提供・運営**
   - アカウントの作成・認証・管理
   - 「今日の質問」の配信および回答の表示
   - タイムライン（フォロー中ユーザーの回答一覧）の生成・表示
   - リアクション機能の提供
   - フォロー・ブロック機能の提供
   - ユーザー検索機能の提供

2. **年齢確認**
   - 13歳以上の利用資格の確認（生年月日を利用）

3. **プッシュ通知の配信**
   - 「今日の質問」の公開通知の送信（デバイストークンを利用）

4. **安全性の確保**
   - NGワードフィルタリングによるコンテンツの品質維持
   - 通報への対応・不正行為の調査
   - アカウント停止（BAN）措置の実施
   - 不正アクセスの検知・防止

5. **サービスの改善**
   - サービスの品質向上・機能改善のための分析

6. **お題管理**
   - ユーザーから提案されたお題の審査・管理

## 4. 情報の保存と管理

### 4.1 保存場所

お客様の情報は、Amazon Web Services（AWS）の東京リージョン（ap-northeast-1）に保存されます。

| データ種別 | 保存先 |
|---|---|
| ユーザー情報・投稿・フォロー関係等 | Amazon DynamoDB |
| プロフィール画像 | Amazon S3 |
| 認証情報（メール・パスワード） | Amazon Cognito |

### 4.2 セキュリティ対策

当社は、お客様の情報を保護するため、以下のセキュリティ対策を講じています。

- パスワードは暗号化された状態で保存されます（Amazon Cognito による管理）
- 通信はHTTPSにより暗号化されます
- APIアクセスには認証トークンが必要です
- 管理者のアクセスは最小権限の原則に基づき制限されています
- 管理者の操作は監査ログに記録されます

### 4.3 保存期間

- **アカウント情報**: アカウントが存在する間保存されます
- **投稿・リアクション等**: アカウントが存在する間保存されます
- **管理者操作ログ**: 90日間保存後、自動的に削除されます

当社は、以下の場合を除き、お客様の個人情報を第三者に提供しません。

1. **お客様の同意がある場合**
2. **法令に基づく場合**: 裁判所、検察庁、警察等の公的機関から法令に基づく要請を受けた場合
3. **生命・身体・財産の保護に必要な場合**: お客様または第三者の生命、身体、財産の保護のために必要であり、お客様の同意を得ることが困難な場合
4. **サービス提供に必要な業務委託先**: サービスの運営に必要な範囲で、以下の業務委託先にデータ処理を委託する場合があります

## 6. お客様の権利

お客様は、ご自身の個人情報について以下の権利を有します。

### 6.1 アクセス権
お客様は、本アプリ内で自身のプロフィール情報、過去の投稿履歴を確認することができます。

### 6.2 訂正権
お客様は、本アプリの設定画面から以下の情報を変更できます。
- 表示名
- プロフィール画像
- 自己紹介
- メールアドレス（確認コードによる認証が必要です）
- パスワード
- 閲覧範囲設定（相互フォローのみ / フォロワーまで）

### 6.3 削除権（アカウント削除）
お客様は、本アプリ内からいつでもアカウントの削除を申請できます。アカウント削除時の処理は以下のとおりです。

- **完全物理削除**: すべてのデータ（投稿、リアクション、フォロー関係、ブロック、通報、プロフィール情報、プロフィール画像、認証情報等）が完全に削除されます
- **他ユーザーからの消去**: 他のユーザーの画面からもお客様の情報は完全に消去されます（「退会済み」等の表示は残りません）
- **即時処理**: 猶予期間やクーリングオフ期間はありません。処理開始後の取り消しはできません
- **再登録**: 退会後、同一のメールアドレスで再度アカウントを登録することは可能ですが、以前のデータは一切復元されません

### 6.4 個人情報に関するお問い合わせ
上記以外の個人情報に関するお問い合わせ（開示請求、利用停止請求等）については、第10条記載のお問い合わせ窓口までご連絡ください。

## 7. 13歳未満のお子様について

本アプリは13歳以上の方を対象としています。当社は、13歳未満のお子様から故意に個人情報を収集することはありません。

登録時に生年月日の入力を必須としており、13歳未満と判定された場合はアカウントの作成ができません。

万が一、13歳未満のお子様がアカウントを登録していることが判明した場合、当社は速やかに当該アカウントおよび関連する個人情報を削除します。お子様が本アプリを利用していることにお気づきの保護者の方は、お問い合わせ窓口までご連絡ください。
