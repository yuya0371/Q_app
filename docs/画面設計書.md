# 画面設計書 — Q.（仮）

## 1. 概要

- **総画面数**: 23画面
- **ナビゲーション**: ボトムタブ（3タブ）+ スタックナビゲーション
- **テーマ**: ダークモード対応（OS設定連動）
- **カラー**: メイン `#1A1A1A`（黒）、アクセント `#8B5CF6`（パープル）

---

## 2. ナビゲーション構造

```
App
├── Auth Stack（未ログイン時）
│   ├── Splash
│   ├── Login
│   ├── Signup
│   ├── EmailConfirmation
│   └── ForgotPassword
│
├── Onboarding Stack（初回設定時）
│   ├── SetAppId
│   └── SetProfile
│
└── Main Tab（ログイン後）
    ├── Home Tab
    │   ├── Home（今日の質問 + タイムライン）
    │   ├── AnswerModal（回答投稿）
    │   ├── QuestionSubmitModal（お題提案）
    │   └── UserProfile（他人のプロフィール）
    │
    ├── Search Tab
    │   ├── Search（ユーザー検索）
    │   └── UserProfile（他人のプロフィール）
    │
    └── MyPage Tab
        ├── MyProfile（自分のプロフィール）
        ├── EditProfile（プロフィール編集）
        ├── AnswerHistory（過去の回答履歴）
        ├── FollowingList（フォロー一覧）
        ├── FollowersList（フォロワー一覧）
        ├── Settings（設定）
        ├── VisibilitySettings（閲覧範囲設定）
        ├── BlockList（ブロック一覧）
        ├── DeleteAccount（アカウント削除）
        ├── ShareProfile（QRコード/共有）
        └── ReportModal（通報）
```

### 3.1 認証系

#### 3.1.1 スプラッシュ画面（Splash）

**目的:** アプリ起動時のローディング表示、自動ログイン判定

**レイアウト:**
```
┌─────────────────────────────┐
│                             │
│                             │
│                             │
│            [ロゴ]            │
│             Q.              │
│                             │
│                             │
│         [ローディング]        │
│                             │
│                             │
└─────────────────────────────┘
```

**構成要素:**
- アプリロゴ（中央）
- ローディングインジケーター

**動作:**
- トークン確認 → 有効ならMain Tab、無効ならLogin画面へ遷移
- 強制アップデートチェック → 必要ならアップデート促進ダイアログ

**構成要素:**
| 要素 | 種類 | 説明 |
|---|---|---|

- メールアドレス: 形式チェック
- パスワード: 入力必須

- 入力欄の下にインラインエラー
- 認証失敗時はトースト通知

## 4. コンポーネント一覧

### 4.1 共通コンポーネント

| コンポーネント | 説明 |
|---|---|
| Button | プライマリ/セカンダリ/デンジャーボタン |
| TextInput | テキスト入力、バリデーション表示対応 |
| IconButton | アイコンのみのボタン |
| Avatar | ユーザーアイコン（サイズ: S/M/L） |
| Card | 汎用カードコンテナ |
| Header | 画面ヘッダー（タイトル、戻るボタン等） |
| TabBar | ボトムタブナビゲーション |
| Modal | モーダル/ボトムシート |
| Toast | 通知トースト |
| LoadingIndicator | ローディング表示 |
| EmptyState | 空状態の表示 |

### 4.2 ドメイン固有コンポーネント

| コンポーネント | 説明 |
|---|---|
| QuestionCard | 今日の質問表示カード |
| AnswerCard | 回答表示カード（タイムライン用） |
| UserListItem | ユーザー一覧の1行 |
| ReactionBar | リアクションボタン群 |
| AnswerHistoryItem | 過去回答の1行 |
